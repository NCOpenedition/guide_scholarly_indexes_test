BASE

BASE

Indexe les métadonnées de toutes sortes de ressources scientifiquement pertinentes - revues, entrepôts institutionnels, collections numériques, etc. - qui disposent d'une interface OAI et utilisent l'OAI-PMH pour fournir leur contenu.

PROCESSUS D'INCLUSION

Element Description
Options d'inclusion Deux options :

1. Suggestion d'un entrepôt OAI-PMH via un formulaire en ligne sur le site web de BASE

2. Collecte automatisée des entrepôts OAI-PMH répertoriés dans les répertoires d'entrepôts OAI
Processus de collecte des données Moissonnage automatisé des entrepôts OAI

Avec ou sans déclaration des administrateurs de bases de données à BASE (collecte automatisée)
