À propos

Ce dépôt contient le Guide de référencement des revues scientifiques, un catalogue de documentation synthétique décrivant le fonctionnement des principales bases de données de revues scientifiques. Il a pour objectif d'aider les éditeurs à choisir les bases de données appropriées pour leurs publications ainsi qu'à mieux comprendre les critères requis pour l'inclusion de leurs articles.

Le guide à été élaboré dans le cadre du projet CRAFT-OA financé par Horizon Europe, dont l'objectif est de contribuer à la construction du European Diamond Capacity Hub.

Le European Diamond Capacity Hub (EDCH) et le projet CRAFT-OA visent à soutenir les éditeurs et les auteurs dans le respect des principes du modèle d'accès diamant. Le Guide de référencement des revues scientifiques constitue l'un des volets de cette initiative.
