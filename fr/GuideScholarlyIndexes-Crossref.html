Crossref

Crossref est une organisation regroupant 19 000 membres issus de l'édition commerciale et institutionnelle. Son principal service consiste à relier les publications qui sont citées et qui en citent d'autres. Les éditeurs associés enregistrent un DOI Crossref et soumettent les métadonnées de leurs propres publications.

PROCESSUS D'INCLUSION

Element Description
Options d'inclusion L'adhésion est obligatoire pour s'inscrire auprès de Crossref. Elle est ouverte aux éditeurs et aux fournisseurs de services.



Les membres peuvent ensuite attribuer des DOI Crossref pour leurs utilisateurs. La génération des DOI peut varier d'un service à l'autre : automatique ou à la demande, pour tout contenu ou pour un contenu spécifique, etc.
Processus de collecte des données Outils dédiés à Crossref :

- Plugin XML Crossref pour OJS (Open Journal Systems)

- Formulaire de dépôt en ligne

- Formulaire d'enregistrement de subvention (pour enregistrer uniquement les métadonnées de la subvention)



Il est également possible d'envoyer des fichiers XML conformes au schéma XSD de Crossref :

- Transmettre des fichiers XML JATS via le formulaire de dépôt en ligne

- Transmettre des fichiers XML via notre outil d'administration

- Dépôt XML via HTTPS POST
Formulaire pour les fournisseurs de données https://www.crossref.org/_apply/member/
Informations de contact https://support.crossref.org/hc/en-us/requests/new?ticket_form_id=360001642691 (formulaire Web)
