# BASE

Indexe les métadonnées de toutes sortes de ressources académiquement pertinentes - revues, référentiels institutionnels, collections numériques, etc. - qui fournissent une interface OAI et utilisent OAI-PMH pour fournir leur contenu.

| Element | Description |
| :- | :- |
