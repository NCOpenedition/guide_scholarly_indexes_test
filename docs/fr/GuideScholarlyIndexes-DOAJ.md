# DOAJ

Le DOAJ (Directory of Open Access Journals) est un répertoire de revues en libre accès à comité de lecture. Il est géré par la communauté et maintenu par Infrastructure Services for Open Access (IS4OA). Le DOAJ permet également la collecte et l'affichage des métadonnées des articles, ainsi que la diffusion gratuite des métadonnées des articles et des revues.

## PROCESSUS D'INCLUSION

| Element | Description |
| :- | :- |
| Options d'inclusion | Les éditeurs doivent s'inscrire et remplir un formulaire de candidature en ligne : https://doaj.org/account/login?redirected=apply <br><br>L'admission nécessite le respect d'un ensemble de critères éditoriaux et techniques. |
| Processus de collecte des données  | Les informations relatives à la revue sont enregistrées lors de la soumission, en premier lieu via un formulaire web.<br><br>Une fois la soumission validée, les informations de l'article peuvent être importées et mises à jour de différentes manières :<br>- Manuellement sur le site web du DOAJ, à l'aide d'un formulaire dédié à la saisie des métadonnées ;<br>- Via l'API du DOAJ, en utilisant des fichiers JSON ;<br>- En téléversant des fichiers au format XML natif du DOAJ : https://www.doaj.org/static/doaj/doajArticles.xsd <br>- Ou en téléversant des fichiers suivant les schémas Crossref : Crossref 5.3.1 XML (https://www.doaj.org/static/crossref/crossref5.3.1.xsd); Crossref 4.4.2 XML (https://www.doaj.org/static/crossref/crossref4.4.2.xsd)/<br><br>Des informations détaillées sur ces procédures sont disponibles ici : https://doaj.org/docs/faq/  |

## CONDITIONS MINIMALES

| Element | Description |
| :- | :- |
| Critères éditoriaux | Revues en libre accès<br>Déclaration d'accès ouvert<br>Licence ouverte<br>Texte intégral disponible sans embargo<br>Tous les domaines de recherche<br>Publication d'au moins cinq articles de recherche par an<br>Lectorat principal : chercheurs et praticiens<br><br>Les revues nouvelles ou récemment passées en accès ouvert doivent justifier d'un historique de publication de plus d'un an ou avoir publié au moins dix articles de recherche en libre accès. |
| Formats de données | HTML ou PDF |
| Standards de métadonnées | Aucun |
| Formats de métadonnées | XML, JSON, formulaire Web (text) |
| Métadonnées minimales | Titre de la revue<br>ISSN ou e-ISSN<br>Date de publication<br>Titre de l'article<br>URL du texte intégral |
