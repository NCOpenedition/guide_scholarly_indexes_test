# Dimensions

Dimensions est une base de données contenant des résumés, des citations et des mentions de subventions de recherche, qui relie ces subventions aux publications, essais cliniques et brevets qui en résultent. Dimensions fait partie du Digital Science (ou Digital Science & Research Solutions Ltd), une entreprise technologique basée à Londres, au Royaume-Uni.

## PROCESSUS D'INCLUSION

| Element | Description |
| :- | :- |
| Options d'inclusion | L'inclusion s'effectue en rejoignant l'une des sources de données mentionnées dans les sources d'index. |
| Processus de collecte des données  | Collecte de données provenant d'autres bases de données.<br><br>Les données sont converties en un modèle de données commun, nettoyées, puis enrichies afin d'être prêtes à l'emploi. L'enrichissement comprend la désambiguïsation des personnes (« Chercheurs ») et des organisations, ainsi que la catégorisation des données par thèmes (« Catégories ») grâce à une classification algorithmique basée sur l'apprentissage automatique. |
